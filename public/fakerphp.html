Fakerphp Lorem Toneflix Images â€‹

Fakerphp Lorem Toneflix Images is an alternative image provider for fakerphp using images from Lorem Toneflix

Installation â€‹

Install the package using composer:

bash
composer require --dev toneflix-code/fakerphp-lorem-toneflix

Usage â€‹

Add Provider â€‹

php
$faker = \Faker\Factory::create();
$faker->addProvider(new \ToneflixCode\FakerLoremToneflix\FakerLoremToneflixProvider($faker));

Return Image URL â€‹

php
$image = $faker->imageUrl();

Set Category â€‹

php
$image = $faker->imageUrl(category: 'avatar');

Set Image Resolution â€‹

php
$image = $faker->imageUrl(width: 800, height: 600);

Pixelate â€‹

php
$image = $faker->imageUrl(pixelate: 5);

Greyscale â€‹

php
$image = $faker->imageUrl(grey: true);

Add Text â€‹

php
$image = $faker->imageUrl(text: 'I Love You');

Marshup â€‹

php
$image = $faker->imageUrl(
    grey: true,
    text: 'I Love You',
    width: 800,
    height: 600,
    pixelate: 5,
    category: 'avatar'
);
